## Design Context

### Users
Crypto traders and researchers running an automated AI trading agent (the "Allele" engine) on Polygon. They want to monitor their trading capital and bot status, but don't want to get bogged down in low-level blockchain infrastructure details like faucets or RPCs.

### Brand Personality
Modern, polished, professional, and seamless. It should feel like a high-end trading terminal or a sophisticated tech product, not a chaotic crypto experiment.

### Aesthetic Direction
High-quality frontend execution taking heavy inspiration from 21st.dev. Dark mode UI, technical but clean. We want to abstract away crypto jargon (e.g. calling testnet "Simulation Mode" and test funds "Practice Credits").

### Design Principles
1. Abstract crypto infrastructure behind clean, intuitive metaphors.
2. Fast, optimistic interactions with progressive disclosure.
3. Clean typography and subtle spatial rhythm. No "AI slop" or generic maximalist gradients unless explicitly purposeful.
4. Meticulous refinement in every component detail.
